Image Processing SciKit¶

The scikits-image SciKit (toolkit for SciPy) extends scipy.ndimage to provide a versatile set of image processing routines. It is written in the Python language.

This SciKit is developed by the SciPy community. All contributions are most welcome! Please join us on the mailing list (address provided below).
