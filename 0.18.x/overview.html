scikit-image: image processing in Python¶

scikit-image builds on scipy.ndimage to provide a versatile set of image processing routines in Python.

This library is developed by its community, and contributions are most welcome!

Read about our mission, vision, and values and how we govern the project.

Major proposals to the project are documented in SKIPs.
